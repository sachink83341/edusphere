1. Algebra

1.1 Linear Equations

A linear equation is an equation of the form ax + b = 0, where a and b are constants and a ≠ 0.

Standard Form:

ax + b = 0

Solution: x = -b/a

Example:

Solve: 2x + 6 = 0

Solution: 2x = -6

x = -6/2 = -3

1.2 Quadratic Equations

A quadratic equation is an equation of the form ax² + bx + c = 0, where a ≠ 0.

Quadratic Formula:

x = (-b ± √(b² - 4ac)) / 2a

Discriminant: Δ = b² - 4ac

  • • If Δ > 0: Two real distinct roots
  • • If Δ = 0: One real root (repeated)
  • • If Δ < 0: No real roots

Example:

Solve: x² - 5x + 6 = 0

Using factoring: (x - 2)(x - 3) = 0

Solutions: x = 2 or x = 3

1.3 Polynomials

A polynomial is an expression consisting of variables and coefficients involving operations of addition, subtraction, multiplication, and non-negative integer exponents.

Types of Polynomials:

  • • Monomial: Single term (e.g., 3x²)
  • • Binomial: Two terms (e.g., x + 2)
  • • Trinomial: Three terms (e.g., x² + 2x + 1)
  • • Degree: Highest power of the variable

2. Geometry

2.1 Basic Geometric Shapes

Circle

  • • Area = πr²
  • • Circumference = 2πr
  • • Diameter = 2r

Rectangle

  • • Area = length × width
  • • Perimeter = 2(l + w)
  • • Diagonal = √(l² + w²)

Triangle

  • • Area = ½ × base × height
  • • Perimeter = a + b + c
  • • Heron's formula: A = √s(s-a)(s-b)(s-c)

Square

  • • Area = side²
  • • Perimeter = 4 × side
  • • Diagonal = side × √2

2.2 Coordinate Geometry

Coordinate geometry deals with the study of geometry using coordinates.

Important Formulas:

  • • Distance between two points: d = √[(x₂-x₁)² + (y₂-y₁)²]
  • • Midpoint: M = ((x₁+x₂)/2, (y₁+y₂)/2)
  • • Slope of a line: m = (y₂-y₁)/(x₂-x₁)
  • • Equation of a line: y = mx + c

3. Calculus

3.1 Limits

A limit describes the behavior of a function as its input approaches a particular value.

Limit Notation:

lim(x→a) f(x) = L

This reads as "the limit of f(x) as x approaches a equals L"

Important Limit Rules:

  • • lim(x→a) c = c (constant)
  • • lim(x→a) x = a
  • • lim(x→a) [f(x) + g(x)] = lim(x→a) f(x) + lim(x→a) g(x)
  • • lim(x→a) [f(x) × g(x)] = lim(x→a) f(x) × lim(x→a) g(x)

3.2 Derivatives

The derivative of a function represents the rate of change or slope of the function at any point.

Derivative Definition:

f'(x) = lim(h→0) [f(x+h) - f(x)]/h

Common Derivatives:

  • • d/dx(c) = 0
  • • d/dx(x) = 1
  • • d/dx(xⁿ) = nx^(n-1)
  • • d/dx(sin x) = cos x
  • • d/dx(cos x) = -sin x
  • • d/dx(eˣ) = eˣ
  • • d/dx(ln x) = 1/x

3.3 Integration

Integration is the reverse process of differentiation. It's used to find areas, volumes, and accumulated quantities.

Basic Integration Rules:

  • • ∫ c dx = cx + C
  • • ∫ x dx = x²/2 + C
  • • ∫ xⁿ dx = x^(n+1)/(n+1) + C (n ≠ -1)
  • • ∫ sin x dx = -cos x + C
  • • ∫ cos x dx = sin x + C
  • • ∫ eˣ dx = eˣ + C

50th percentile

Most frequently occurring value

Highest frequency

4.2 Probability

Probability is the measure of the likelihood of an event occurring.

Basic Probability Formula:

P(A) = Number of favorable outcomes / Total number of outcomes

Where 0 ≤ P(A) ≤ 1

Probability Rules:

  • • P(A ∪ B) = P(A) + P(B) - P(A ∩ B)
  • • P(A') = 1 - P(A)
  • • P(A ∩ B) = P(A) × P(B|A)
  • • For independent events: P(A ∩ B) = P(A) × P(B)

Important Formulas Reference

Algebraic Identities

  • • (a + b)² = a² + 2ab + b²
  • • (a - b)² = a² - 2ab + b²
  • • a² - b² = (a + b)(a - b)
  • • (a + b)³ = a³ + 3a²b + 3ab² + b³
  • • (a - b)³ = a³ - 3a²b + 3ab² - b³
  • • a³ + b³ = (a + b)(a² - ab + b²)
  • • a³ - b³ = (a - b)(a² + ab + b²)

Trigonometric Identities

  • • sin²θ + cos²θ = 1
  • • tan θ = sin θ / cos θ
  • • 1 + tan²θ = sec²θ
  • • 1 + cot²θ = csc²θ
  • • sin(A + B) = sin A cos B + cos A sin B
  • • cos(A + B) = cos A cos B - sin A sin B
  • • sin 2θ = 2 sin θ cos θ
